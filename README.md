# Scripts

![License](https://img.shields.io/badge/license-MIT-blue)
![Languages](https://img.shields.io/badge/languages-PowerShell%20%7C%20JavaScript-blue)

A collection of PowerShell scripts and userscripts for system administration, debloating, network management, security hardening, and browser customization.

## Categories

### PowerShell
- **Adobe** — Adobe product management and cleanup scripts
- **Debloat** — Windows bloatware removal and optimization
- **Network** — Network configuration and diagnostics
- **Security** — System hardening and security audit tools
- **VoidTools Everything** — Configuration and management scripts

### Userscripts
- Browser enhancement and customization scripts for various websites

## Usage

Each script is standalone. Navigate to the relevant category folder and run the script you need.

```powershell
# Example: Run a debloat script
.\Powershell\Debloat\ScriptName.ps1
```
